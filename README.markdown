# Setup-linux

Installation des fichiers pour configurer mon linux.

L'installation comprend :
* l'installation de paquets par défaut pour linux
* l'installation de paquets pour EIME ainsi que le script CODDE

Pour l'installation taper :

```
$ wget https://raw.github.com/thooams/setup-linux/master/install.rb
$ chmod a+x install.sh
$ sudo ./install.sh
```

Installer le theme gnome terminal tomorrow-night
 ```
$ chmod a+x  setup-gnome-terminal-theme.sh
$ ./setup-gnome-terminal-theme.sh
```
